# Предсказание покупательской активности

## Постановка задачи

Интернет-магазин «В один клик» столкнулся со снижением активности покупателей. Задача — построить модель бинарной классификации, которая предскажет вероятность снижения покупательской активности клиента в следующие три месяца. На основе модели и данных о прибыльности клиентов необходимо выделить сегменты покупателей и разработать персонализированные предложения.

**Целевой признак**: `Покупательская активность` — «снизилась» или «прежний уровень».

## Данные

| Файл | Описание |
|------|----------|
| `market_file.csv` | Поведение на сайте, коммуникации, продуктовое поведение |
| `market_money.csv` | Выручка по периодам |
| `market_time.csv` | Время на сайте по периодам |
| `money.csv` | Среднемесячная прибыль магазина по клиентам |

## Используемые модели

- **KNeighborsClassifier** — метод k ближайших соседей
- **DecisionTreeClassifier** — дерево решений
- **LogisticRegression** — логистическая регрессия
- **SVC** — метод опорных векторов

Для обработки данных использованы пайплайны (`Pipeline`) с `ColumnTransformer`, включающие различные кодировщики (`OrdinalEncoder`, `OneHotEncoder`) и скейлеры (`StandardScaler`, `MinMaxScaler`). Гиперпараметры подобраны с помощью `GridSearchCV`.

## Результаты

**Метрика**: ROC-AUC (оптимальна для несбалансированных классов и задач ранжирования по вероятности).

**Лучшая модель**: LogisticRegression с ROC-AUC ≈ 0.91 на тестовой выборке.

Анализ важности признаков (SHAP) показал, что наибольшее влияние на целевой признак оказывают:
- Длительность регистрации на сайте
- Среднее число просматриваемых категорий за визит
- Маркетинговая активность за 6 месяцев

**Сегмент для персонализации**: клиенты с высокой вероятностью снижения активности и высокой прибыльностью — приоритетная группа для индивидуальных предложений и усиленной коммуникации.
